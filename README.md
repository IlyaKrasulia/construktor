# Construktor

A **mobile-first React SPA** — catalog shop for 50+ constructor sets, built with **Vite** and **Tailwind CSS**.

## Features

- 📱 **Mobile-first design** — product cards in a 2-column grid
- 🧱 **12 mock constructor products** with images (picsum.photos), name, price and description
- 📌 **Sticky header** with brand logo and cart icon (with item counter badge)
- 🪟 **Bottom Sheet modal** — slide-up detail view on card tap with smooth animation
- 🛒 **"Buy in 1 click"** prominent button on every card and in the modal
- 🎨 **Minimalist style** — white space, `rounded-2xl`, indigo accent colour
- 🔔 **Toast notifications** for cart additions

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```
